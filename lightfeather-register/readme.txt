The registration challenge could be done with simple html, javascript, and CSS but I chose a modern web
technology: Angular. For a nice look and feel and used Angular Material which is an Angular implementation
of the Material Design UI template.

To run this app

1. Install NodeJs onto your system
2. Checkout code from Github
3. In the root of the project, execute "npm install"
4. Then execute "ng server"
5. In a browser go to  http://localhost:4200
